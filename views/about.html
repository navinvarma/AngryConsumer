Angry Consumer!

"Angry Consumer!" is a free web application ("app") that is provided as a service to consumers of financial products in the United States of America. This application provides an alternative user experience to the existing Consumer Finance Compliant Database.

This app provides consumers of financial products the ability to:

  • Search various complaints submitted to the Consumer Finance Protection Bureau.
  • List consumer complains by Company name.
  • Visualize consumer complaints grouped into various categories for data analysis.
  • REST API endpoints to request data in a JSON format that is presented in the web application
For more information on the dataset used in this application, visit Data.gov.

Privacy Policy

"Angry Consumer!" does not collect any information other than error log messages captured in the hosted virtual private server. All data provided by this web application is from publicly available data published by the Consumer Finance Protection Bureau.

Use of Cookies

"Angry Consumer!" does not use any cookies.
